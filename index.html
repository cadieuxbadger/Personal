MY WORK

  • Makeup Matchr: A 6-week, 5-person Capstone Design Project to make a mobile-friendly solution to finding cosmetic foundation that best matches one's skin tone. This was made in Django (Python/HTML/SQL) and utilized a scratch-made color picker, KNN-style matching algorithm, and an Adobe-developed color correction algorithm. Originally hosted on https://matchr.makeup, but has been taken down since the end of the project. (Github Repository)

  • Pet Stop: A 4-week, 4-person Database Systems Project to learn relational databases and SQL. We used XAMPP (PHP/HTML/SQL) to locally host this project, meant to match pet owners - which had their pets linked to their accounts - with pet sitters - who listed which types of pets they'd like to watch. This took each users' calendars and pet types into account to find matching user, and presented pet owners with a list of available sitters, along with reviews about each one. (Github Repository)
